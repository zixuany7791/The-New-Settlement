# The New Settlement

### Description:

You have found a new land after weeks of traveling. Your goal is to survive in this land with your crew and build a thriving city. 

### Controls:

W - Move Up 

S - Move Down 

A - Move Left

D - Move Right 

E - Interact with structures 

X - Menu for building 

Escape - Exit menu
